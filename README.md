## 📌 Sobre o Repositório
Este repositório contém os projetos, códigos e anotações desenvolvidos durante os meus estudos no curso **Spring Boot Direto Das Trincheiras**, da plataforma **DevDojo** (ministrado por William Suane).

O objetivo deste repositório é documentar a aplicação de conceitos avançados e cenários reais do mercado de trabalho, focando em como o Spring Boot é utilizado em produção em ambientes de alta exigência.

## 🎯 Foco e Objetivos de Aprendizado
Ao longo deste projeto, o foco está em ir além do básico (CRUD), aplicando:
* **Práticas de Mercado:** Soluções arquiteturais utilizadas no dia a dia de grandes empresas.
* **Clean Code e SOLID:** Escrita de código limpo, manutenível e com responsabilidades bem definidas.
* **Profundidade no Framework:** Entendimento dos bastidores do Spring Boot para resolução de problemas complexos e otimização de performance.
* **Preparação Técnica:** Domínio de conceitos avançados frequentemente abordados em entrevistas para vagas de desenvolvedor Backend.

## 🚀 Tecnologias e Ferramentas Utilizadas (atualizando conforme o andamento das aulas)
* **Java 21+**
* **Spring Boot 3**
* **Gerenciamento de Dependências:** Maven
* **Versionamento:** Git e GitHub
* **MVC**
* **Postman**
